# Browser AI Assistant

A browser extension that allows users to highlight text on any webpage and instantly receive AI-generated answers without leaving the page.

## Features
- Highlight any text on a webpage
- Get instant AI-generated responses directly in the browser
- No need to switch tabs or search manually
- Lightweight and fast user experience

## Tech Stack
- JavaScript
- HTML/CSS
- Browser APIs (text selection, content scripts)
- OpenAI API (AI responses)

## How it Works
The extension captures user-selected text on a webpage and sends it to an AI model, which generates a response that is displayed directly within the browser interface.

## Use Case
Designed to improve productivity by reducing context switching — users can quickly get explanations, summaries, or answers without interrupting their workflow.

## Status
Personal project / prototype

## Setup

To use this extension, you need to insert your own OpenAI API key in the code in Background.js line 1 .

Note: For security reasons, do not expose your API key publicly.
